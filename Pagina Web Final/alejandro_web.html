Mis Marcadores y Sitios de Interés

Wikipedia

La mayor enciclopedia de Internet. Se edita de manera colaborativa, es decir, la propia comunidad es la que se encarga de escribir y actualizarla constantemente. Si buscas información sobre cualquier tema, este es tu lugar.

GitHub

Github es el repositorio más usado de Internet. Puedes encontrar el código de toda clase de proyectos y aplicaciones realizado por todo tipo de personas. Es una web muy útil para aprender a programar y para subir tus propios proyectos y mostrárselo al resto de usuarios.

Twitch

La plataforma de streaming más popular del mundo. Puedes encontrar toda clase de entretenimiento de forma gratuita. Además, también existe la posibilidad de aprender, pues existen canales que emiten contenido didáctico sobre diversas materias.

Youtube

Similar a Twitch. Puedes encontrar cualquier tipo de contenido y aprender muchísimo. A mi personalmente, me ha ayudado en incontables ocasiones para estudiar asignaturas como cálculo, álgebra, física e incluso programación. Además de eso, es mi plataforma principal para escuchar música y entretenimiento.

Metacritic

Web de reviews sobre videojuegos, películas, series y música. Es muy útil para conocer la opinión popular de la gente y estar al tanto de los últimos lanzamientos.

Twitter

El periódico del siglo XXI. Si quieres enterarte de las últimas noticias y estar al tanto de la actualidad, es la mejor aplicación posible. Además, también es una fuente de entretenimiento y puede servir para socializar y conocer gente.
